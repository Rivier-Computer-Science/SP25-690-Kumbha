
## Dataset Setup

The project uses CIFAR-10 dataset. Download it once and extract inside the `data` folder:

1. Download: https://www.cs.toronto.edu/~kriz/cifar-10-python.tar.gz
2. Extract the tar.gz file inside the `data/` folder (it will create `cifar-10-batches-py`).
3. Do not commit the `data/` folder to GitHub.
